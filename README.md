# MILK BOTTLE

A simple milk bottle tracker.

## Features

- Charge milk manually
- Auto recovery (+1 milk every 24 hours)
- Activity logs
- Achievement counters
- Local storage support
- Works on mobile browsers

## Status

Personal project.
